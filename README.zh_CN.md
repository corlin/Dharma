# Dharma (行纪)

> 觉悟而行，在行动中觉醒。

## 🌌 核心哲学 (Philosophy)

**Dharma (法/达摩)** 在梵语中意为“法则、天命、正道”，是瑜伽哲学的核心。它对应道教的“得道”、基督教的“救赎”以及玄学中的“超脱轮回”。

在现代语境下，Dharma 是一张逃离熵增的地图。我们的生命往往受困于混沌、焦虑与无序之中，而本系统的宗旨，就是通过严密的规划与执行，让生命从无序中解脱，重获自由。

**中文名“行纪”寓意**：
*   **行**：行动、修行。不尚空谈，唯有践行。
*   **纪**：法纪、世纪。在混乱的时间流中建立秩序。

系统建立在 5 个基础层级之上：

1.  **Excavate (挖掘/过去)**：直面你的“反愿景”（个人地狱），通过苏格拉底式的质询挖掘你真正的核心价值观。
2.  **Orient (定向/未来)**：确立你的北极星。将你的野心构建成连贯的目标金字塔（十年 → 年度 → 季度 → 本周）。
3.  **Execute (执行/当下)**：进入“苦行模式（Monk Mode）”。高强度的深度工作会话，强制的心流状态，以及完全的干扰屏蔽。
4.  **Feedback (反馈/循环)**：控制论感知。分析“言”（计划）与“行”（实际）之间的差距，通过残酷的诚实进行修正。
5.  **Evolve (进化/身份)**：游戏化成长。通过一致性赚取 XP，升级你的身份等级，将新行为固化为永久的性格特质。

## 🛠 技术栈

为未来的 iOS 开发标准而构建。

-   **平台**: iOS 26+ (Swift 6 并发模型)
-   **UI 框架**: SwiftUI (声明式，动效驱动)
-   **架构模式**: **TCA** (The Composable Architecture) v1.x
-   **数据持久化**: **SwiftData** (Schema 驱动, CloudKit 同步)
-   **智能引擎**: **Local-First AI** (Ollama/CoreML) + 云端回退 (OpenAI/Deepseek)
-   **设计系统**: 定制的“神秘主义毛玻璃（Mystical Glassmorphism）”美学

## ✨ 核心功能

-   **反愿景工坊 (Anti-Vision Workshop)**：通过“逆向思考”找到人生方向的引导式工坊。
-   **目标金字塔 (Goal Pyramid)**：一个可缩放的触觉界面，连接从十年愿景到本周里程碑的每一个环节。
-   **深度工作计时器 (Deep Work Timer)**：带有心流追踪、恢复模式和强制执行机制的专注工具。
-   **苏格拉底 AI 教练 (The Orb)**：一个集成的 AI 实体，它会质疑你的借口，帮助你弥合“言行差距”。
-   **XP 与等级系统**：严格的角色扮演系统，信守承诺获得 XP，背叛誓言则受到惩罚。

## 🚀 快速开始

### 前置要求
-   Xcode 16.0+
-   iOS 18.0+ 模拟器或真机
-   (可选) 本地运行 Ollama 以启用完全隐私的 AI 功能

### 安装步骤
1.  克隆仓库：
    ```bash
    git clone https://github.com/corlin/Dharma.git
    ```
2.  打开 `Dharma.xcodeproj`。
3.  确保选择了正确的签名团队 (Signing Team)。
4.  构建并运行 (⌘+R)。

## 🎨 设计系统

Dharma 采用了一套专注于“审美升维”的设计语言：
-   **排版**: 圆润的人文主义字体 (`.dharmaTitle`, `.dharmaBody`)。
-   **色调**: 深邃虚空 (`.backgroundPrimary`)，电光蓝 (`.execute`)，与神秘紫 (`.evolve`)。
-   **布局**: 极致的留白缩减，追求最高的信息密度与沉浸感。
